# 教大签证进度助手

这是一个本地运行的 Chrome／Edge 扩展。学生正常登录香港教育大学申请系统后，扩展会在官网页面的登录会话中读取签证申请资料，并在独立仪表板中以中文显示。

> 本项目不是香港教育大学或香港入境事务处的官方产品。签证资料与下载结果均以官方网站为准。

## 为什么不能只做成普通网站？

`pappl.eduhk.hk` 的申请资料受登录 Cookie、XSRF 验证和浏览器同源策略保护。部署在 GitHub Pages 或其他域名上的普通网页不能直接读取该登录会话。

本项目采用浏览器扩展：请求在学生已经登录的教大页面内执行，仪表板只负责本机展示。它不会要求账号密码，也不会把查询结果上传到第三方服务器。

## 安装

### Windows 快速启动

直接双击 `启动签证助手.cmd`。脚本会使用独立的本机浏览器资料目录加载扩展并打开教大登录页。首次使用需要正常登录；脚本不会读取或保存你的密码。

### 手动安装

1. 下载或克隆本仓库。
2. Chrome 打开 `chrome://extensions`；Edge 打开 `edge://extensions`。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择本项目文件夹。
5. 建议把“教大签证进度助手”固定到浏览器工具栏。

## 使用

1. 打开并正常登录：<https://pappl.eduhk.hk/VMS/admission/applicant/ImmD/submission>
2. 保持该官网页面开启。
3. 点击浏览器工具栏中的扩展图标。
4. 在信息页点击“查询我的签证资料”。

页面会重点显示：

- 入境处申请参考编号 `immdRefNo`
- 提交、核验和最近退回时间
- 申请人与签证类别等资料
- 香港入境处电子签证查询及下载入口

电子签证官方页面：<https://www.gov.hk/tc/residents/immigration/nonpermanent/downloadevisa.htm>

## 隐私与安全

- 不在代码中写死 XSRF token，运行时从当前官网会话中读取。
- 不读取或保存学生的登录密码。
- 不把接口资料发送到任何第三方服务器。
- 不建议把真实姓名、证件号码、Cookie、XSRF token 或完整接口返回提交到 Git 仓库。
- 关闭官网会话或退出登录后，扩展无法继续查询。

## 项目文件

```text
manifest.json       扩展配置与最小权限
background.js       查找官网标签页并协调请求
content.js          在官网登录会话中调用接口
index.html          信息仪表板
dashboard.css       页面视觉与响应式布局
dashboard.js        中文资料整理与页面渲染
启动签证助手.cmd    Windows 一键启动入口
```

## 本地开发

修改代码后，在扩展管理页点击本扩展的“重新加载”，再刷新已经打开的教大官网页面。

项目没有第三方运行依赖，不需要 `npm install` 或构建命令。

## 已知限制

- 必须在同一个浏览器中登录官网并运行扩展。
- 教大若调整接口路径、字段或 XSRF 存储方式，扩展需要同步更新。
