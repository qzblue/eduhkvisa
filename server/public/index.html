02项目初衷

项目初衷

把复杂的查询步骤，
整理得更简单。

BlueVisa 的起点很简单：许多同学已经完成材料递交，却不知道 MEEN 开头的编号是什么、在哪里找到，以及拿到编号后应该去哪里下载 eVisa。

作者 Blue 希望用一个清楚、克制、公开透明的工具，把散落在系统中的本人信息整理成下一步能真正使用的指引。它是一项免费公益服务，也会一直保留开源版本。

“让每一位等待来港读书的同学，少一点信息差，多一点安心。— Blue，项目作者

03隐私机制

隐私机制

只做查询，
不保存签证资料。

BlueVisa 采用一次性服务端会话。教大登录字段与查询结果只在当前请求期间存在，不建立签证资料数据库。

即时处理教大登录字段

仅用于当次连接官网，完成后不再保留。

从不持久保存Cookie 与签证结果

不写入 PostgreSQL、文件或应用日志。

网站运行所需账号与登录会话

仅保存邮箱、密码哈希和网站 session，用于多人隔离。

公开可核验源代码与字段白名单

明确排除申请状态、上传文件和完整文件清单。

开源项目

代码公开，欢迎检查。

查询逻辑、字段白名单、部署方式和本机插件均已在 GitHub 公开。欢迎审阅、提出问题或参与改进。
